
# 🛒 Oya Confiserie – Premium Pantry Experience

Bienvenue sur **Oya Confiserie**, une plateforme e-commerce moderne dédiée à la vente de produits agroalimentaires et confiseries en Afrique.
Ce projet met en avant une expérience utilisateur fluide, un design haut de gamme et une adaptation parfaite au contexte local.

---

## 🚀 Objectif du projet

Créer une solution digitale performante pour une alimentation générale afin de :

* Faciliter l’accès aux produits du quotidien
* Améliorer l’expérience d’achat en ligne
* Valoriser les produits locaux africains
* Augmenter les ventes grâce à une interface intuitive et moderne

---

## 🌍 Contexte

Ce projet a été conçu pour le marché africain (Togo), en prenant en compte :

* L’utilisation majoritaire du mobile 📱
* Les connexions internet parfois limitées ⚡
* Les moyens de paiement locaux (Mobile Money, paiement à la livraison)

---

## 🎨 Design & Expérience Utilisateur

* Interface **moderne, épurée et professionnelle**
* Palette : **blanc, noir, violet / bleu premium**
* Typographie : **Poppins**
* Animations fluides pour une expérience immersive

## 🖼️ Fonctionnalités principales

### 🏠 Page d’accueil

* Hero section avec **carrousel dynamique**
* Call-to-action impactants
* Mise en avant des produits populaires

### 🛍️ Catalogue produits

* Produits agroalimentaires :

  * Riz, huile, sucre
  * Boissons
  * Produits locaux africains
* Confiseries :

  * Gâteaux
  * Petits fours

* Image produit
* Description détaillée
* Prix clair
* Bouton **Ajouter au panier**

* Ajout / suppression de produits

### 💳 Commande (Checkout)

* Formulaire simple :

  * Nom
  * Adresse
  * Numéro
* Options :
